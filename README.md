# sayhello
**留言板demo，来源于《Flask Web开发实战》，作者李辉，非常好的一本书。**

####1. 如何下载static静态资源
https://getbootstrap.com/docs/4.2/getting-started/download/
Download source->dist/(css,js)

####前端开发流程：
1. 根据功能规格书画页面草图
2. 根据草图做交互式原型图
3. 根据原型图开发前端页面（HTML、CSS、js）
####后端开发流程：
1. 数据库建模
2. 编写表单类
3. 编写视图函数和相关处理函数
4. 在页面中使用Jinja2替换虚拟数据。
